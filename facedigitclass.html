Face & Digit
Classification

Completed this project as part of the final project for CS 520 - Introduction to Artificial Intelligence. In this project, code for feature extraction of digit and face data was developed, following which, Naive Bayes and Perceptron was successfully implemented from scratch.

Technology Used: Python

Developed code for feature extraction of digit and face data. Successfully implemented Naive Bayes and Perceptron from scratch to achieve an accuracy of 73% (face) and 76% (digit) using Naive Bayes and 83.3% (face) and 80% (digit) using Perceptron. Improved the classification accuracy of face data by 14.7% and digit data by 5.9% by converting extracted features to image pixels using cv2 and implementing a CNN model.
