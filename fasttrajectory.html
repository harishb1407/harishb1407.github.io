Fast Trajectory
Replanning

Completed this project as part of a project for CS 520 - Introduction to Artificial Intelligence. In this project, code for a maze-solving application using the A* search algorithm that enables an agent to navigate a 101x101 grid maze with obstacles to reach a target cell was developed.

Led a team of 3 in developing a maze-solving application using the A* search algorithm that enables an agent to navigate a 101x101 grid maze with obstacles to reach a target cell using the shortest route, with a Manhattan distance heuristic. Implemented and tested Backward A* search and Adaptive A* search algorithms, comparing their execution times to Forward A* search. Created an interactive graphical user interface using the pygame module to visually display the shortest path the agent takes to reach the target.
